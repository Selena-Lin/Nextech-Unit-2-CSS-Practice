# Nextech Catapult Unit 2 CSS Practice
Modified practice activities for quick CSS lesson.

## 🎯 Objectives
- Understand the purpose and structure of CSS.
- Learn how to select and style HTML elements using CSS.
- Gain hands-on experience with various CSS properties and values.

Practice covers the following:
- Changing background color
- Changing font size and text color
- Using class attributes
- Margin, padding, and borders
- Flexboxes
